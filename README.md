# dora-proceedings

Authorised English translations, paired with the Swedish originals, of administrative and legal documents from proceedings on State liability for inadequate application of Regulation (EU) 2022/2554 (DORA) in Sweden.

## Purpose

This repository archives primary-source documents from the proceedings so they can be referenced with a stable identifier. Each Swedish original is paired with an authorised English translation that carries the translator's certification and an apostille under the Hague Convention of 5 October 1961. The Swedish originals are the authoritative texts; the translations are provided for accessibility.

## Contents

- [`justitiekanslern-2026-1950/`](justitiekanslern-2026-1950/) — Claim for damages against the Swedish State before the Chancellor of Justice (Justitiekanslern), case no. 2026/1950, concerning the Swedish Financial Supervisory Authority's (Finansinspektionen) application of DORA. See the folder's README for the document list.

## Translation and certification

All English translations were produced by Timothy Ian Cramp, a translator authorised by the Swedish Legal, Financial and Administrative Services Agency (Kammarkollegiet), translator no. 890. Each translation carries a signed certification page and an apostille issued under the Hague Convention of 5 October 1961.
