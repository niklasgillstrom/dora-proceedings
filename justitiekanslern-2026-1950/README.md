# Justitiekanslern, case no. 2026/1950

Claim for damages against the Swedish State before the Chancellor of Justice (Justitiekanslern), concerning the Swedish Financial Supervisory Authority's (Finansinspektionen, FI) application of Regulation (EU) 2022/2554 (DORA). The claim is brought under the Francovich principle and the Swedish Tort Liability Act (1972:207).

This folder contains FI's opinion in the matter (FI ref. 26-7652) and Gillsoft AB's response to that opinion, each as the Swedish original and an authorised, apostilled English translation.

## Notes on the translation files

Each English translation file begins with the translator's certification page and ends with an apostille under the Hague Convention of 5 October 1961. The reported page count for a translation file therefore includes the certification page and the apostille page in addition to the translated text. The Swedish originals are the authoritative texts.
